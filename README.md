Game Idea

Autobahn Rat is a simple 2D driving game made with Python and Pygame.
You control a yellow car and try to survive as long as possible while avoiding other cars on the road.
The game gets faster over time, so it becomes harder the longer you survive.

How to Play

Move the car up and down to avoid traffic

Crashing into cars reduces health

When health reaches zero, the game ends

Your distance is your score

The game saves your best distance automatically

Controls

W / Up Arrow – Move up

S / Down Arrow – Move down

Classes Used

HighScore – saves and loads the best score

Player – handles player movement

DrawWindow – draws everything on the screen

CrashScreen – shows the crash screen when the game ends

Extra Features

Speed increases over time

Distance counter

High score saving

Cars are drawn with proper depth

Code is organized using classes

Assets

Images were created or generated for this project.
Some visuals were generated using an AI tool for educational purposes.

How to Run

Install pygame:

pip install pygame


Run the game:

python main.py
